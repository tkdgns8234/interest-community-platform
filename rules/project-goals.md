# project-goals
1. 조건
    1. 사람들이 당연하게 생각하는 결제 카테고리를 무료로 풀고, 광고를 붙인다
    2. MAU가 많이 나와야 한다
        1. 광고 수입을 위해 지속적인 체류, 접속을 유도할 수 있어야한다
2. 요세 취미, 관심사 모임 등을 찾으려해도 찾기가 쉽지 않다
    1. 영어공부, 러닝, 게임, 이성친구 만들기등 취미, 관심사 관련 커뮤니티 서비스
        1. 사용자 실명 인증 기반으로 무분별한 나쁜 글, 범죄 등이 발생하지 않는 clean 한 서비스 지향
        2. 당근과 같은 게시글 커뮤니티 기능
            1. 취미 등 카테고리 선택 가능
                1. ex) 아이온2 루드라 클리어파티 가실분
                2. 영어 study 모집 (내국,외국인간 언어 교환 모임, 워홀 정보 공유 등)
        3. 채팅 기능
            1. 채팅 매칭 기능의 차별점
                1. 1:1 매칭이 아니라, 동일한 관심사를 가진 N명이 매칭 돌리기 가능
                    1. 이 N명 채팅 매칭 OR 채팅 방 생성 후 N명 난입 가능하도록 하는 기능이 차별점이 될수도
                        1. ex) UFC 생중계 관련 live 소통방
            2. 관련 카테고리별 채팅매칭기능
                1. text , 화상 채팅(아이폰 처럼 나의 모습을 캐릭터화 가능하도록, 부담줄이기)
            3. 관심사별 랜덤 매칭 기능
                1. 내부에서 진행할 수 있는 모임, 취미는 이 앱 안에서 해결 가능하도록
                    1. 취미 카테고리 선택 가능
                    2. 나이대 선택 가능
                    3. 국가, 지역 선택 가능
                    4. 성별 선택 가능
    2. 앱 내 지속적 접근 및 체류가능하도록 기능 추가 필요함
        1. 카톡 등 외부서비스로 빠지지 않을만한 이유 만들기
            1. 알림 기능
                1. 특정 시간 알림 -> text, 화상 채팅방 모여서 소통 가능하도록 유도
            2. 캘린더 (일정 관리), todo, 채팅방 , 프로필 에서 타 커뮤니티와 차별점 생각해보기
                1. 프로필 -> 요세 이세계아이도르 뜨는것처럼 나라는 캐릭터화
            3. 신규인원 초대 기능
                1. 지속적 추가 인원 매칭 및 게시글 커뮤니티 통해서 채팅방 신규 유입 가능
3. 카카오톡 오픈채팅 서비스 에서 불편했던 점도 우리 서비스로 흡수할 수 있도록 개선점 추가
4. 프로필
    1. 토스증권쪽 더 참조해서 재미요소 붙여보기 
5. 수익 모델
    1. 게시글 커뮤니티 속 광고 (당근처럼)
    2. 앱 내부 광고영역
6. UI는 검정,파랑 톤앤매너

## 위 기능을 전부 현재 단계에서 구현하기에는 무리가 있으므로, MVP 기능을 우선 구현한다.
- MVP 기능
  1. 유저
     * 이메일 / 소셜 로그인
     * 닉네임
     * 실명 인증 여부 (Boolean)
       * 실명 인증 기능은 MVP 단계에서 제외
     * 관심사 카테고리 선택 (최소 1개)
     * 팔로우 기능

  2. 카테고리 & 토픽 
    카테고리 (영어공부)
    └ 토픽 (영어 스터디 모집 글)
     * 카테고리 조회
     * 토픽 조회
     * 토픽 소개글
     * 토픽 게시글 리스트
     * 토픽 캘린더
     
  3. 게시글 + 댓글
     * 게시글 작성
     * 게시글 리스트
     * 댓글 작성
     * 조회수
     * 좋아요
     
  4. 채팅
     * 도메인 설계만 해두고 실제 구현은 MVP 단계에서 제외
     * 1:1 채팅 매칭
     * 채팅방 입장
     * 채팅 메시지 전송/수신
